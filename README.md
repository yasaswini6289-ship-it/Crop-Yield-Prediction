# 🌾 Crop Yield Prediction using Machine Learning

## 📌 Overview
This project predicts crop yield using multiple Machine Learning regression algorithms. It involves data preprocessing, feature encoding, model training, 
evaluation, and comparison to identify the best-performing model. The project is implemented in Python using Jupyter Notebook.

## Features
- Data preprocessing and cleaning
- Label Encoding for categorical variables
- Train-test data splitting
- Multiple regression models
- Performance evaluation using R² Score and Mean Squared Error (MSE)
- Visualization of model performance

- Matplotlib
- Seaborn

## 🤖 Machine Learning Models
The following regression models are implemented:
- Linear Regression
- Support Vector Regression (SVR)
- Decision Tree Regressor
- Random Forest Regressor

## 📊 Evaluation Metrics
Models are evaluated using:
- R² Score
- Mean Squared Error (MSE)
The model with the highest R² Score is selected as the best-performing model.

## ▶️ Usage
1. Download the dataset (`yield_df`) from Kaggle.
2. Place it in the project directory.
3. Open `Crop_Yield_Prediction.ipynb`.
4. Run all notebook cells sequentially.
5. Compare the performance of different regression models.
